## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/10mohamedsadeki/teamflow-saas-dashboard.git
```

Go to the project folder:

```bash
cd teamflow-saas-dashboard
```

Install dependencies:

```bash
npm install
```

Create `.env.local`:

```env
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_key
```

Run the development server:

```bash
npm run dev
```
